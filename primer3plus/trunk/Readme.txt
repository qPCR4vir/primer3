Primer3plus is a webinterface for the primer3 program.

It requires the primer3, PERL and Apache.

It should run unaltered on windows and linux systems.

To install copy the folder in the cgi-folder and adjust 
the settings in settings.pl

All cgi-scripts should be executable by the webserver user.
chmod a+x *.cgi

This user should have write access to the cached_data folder.
